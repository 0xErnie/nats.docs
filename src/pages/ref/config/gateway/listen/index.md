# listen

/ [Server Config](/ref/config/index.md) / [gateway](/ref/config/gateway/index.md) 

`<host>:<port>` format. Alternative to `host`/`port`.

*Reloadable*: `true`

*Types*

- `string`


## Examples

### Hostname
```
localhost:4222
```
### IPv4
```
0.0.0.0:4222
```
### IPv6
```
[::]:4222
```
