# server_tags

/ [Server Config](/ref/config/index.md) 

One or more tags associated with the server. This is currently
used for placement of JetStream streams and consumers.

*Reloadable*: `true`

*Types*

- `string`
- `array(string)`


## Examples

```
cloud:aws
```
```
[region:us-west, az:1b]
```
